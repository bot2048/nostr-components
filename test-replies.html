Nostr Post with Replies Feature

How to use the new replies feature:

  • Attribute Control: Set show-replies="true" to show replies by default
  • Interactive Toggle: Click the "Show Replies" / "Hide Replies" button in the stats section
  • Reply Display: Shows user avatar, name, and reply content only
  • No Stats: Replies don't show individual stats (reactions, zaps, etc.)
